# GaraTrac

A native Android tracker application built with Jetpack Compose using Material 3 design. 
Made for fun over winter break, so nothing too serious here.

## Development

If you wish to see the code for some reason, open the project in Android Studio and sync Gradle project.

## Configuration

To use your own server, you need a backend server with an API like [this](https://github.com/perifural/tracker/blob/main/v2/server.js).
Configure the **Server URL** and **Device ID** in the app settings.
